# Cherry Servers Provider

The Cherry Servers provider is used to interact with the resources supported by [Cherry Servers](https://www.cherryservers.com/). The provider needs to be configured with the proper credentials before it can be used.

Use the navigation to the left to read about the available resources.

## Example Usage

```terraform
terraform {
  required_providers {
    cherryservers = {
      source = "cherryservers/cherryservers"
    }
  }
}

# Set the variable value in *.tfvars file
# or using the -var="cherry_api_key=..." CLI option
variable "cherry_api_key" {
  sensitive = true
}

# Configure the Cherry Servers Provider.
provider "cherryservers" {
  api_token = var.cherry_api_key // API key can be found in Cherry Servers client portal - https://portal.cherryservers.com/settings/api-keys
}
```
